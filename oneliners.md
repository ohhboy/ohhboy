# Oneliners

### Simple Python Server

```text
python -m SimpleHTTPServer 8000

sudo python3 -m http.server 80
```

### Extract tar file

```text
tar -zxvf file.tar.gz
```

### Capture Traffic / Man in the middle

```text
sudo tcpdump -i tun0 icmp
```

### Install deb file

```text
sudo sudo dpkg -i package_file.deb
```
